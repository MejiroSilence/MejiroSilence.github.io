使用anime4K-实时动画超分辨率

发表于2022-10-01|更新于2022-10-01|动画

什么是anime4K

anime4K是github上的一个开源项目

可以实时在观看动画的同时，实时提高动画的分辨率

项目中的作者的简介

Anime4K is a set of open-source, high-quality real-time anime upscaling/denoising algorithms that can be implemented in any programming language.

The simplicity and speed of Anime4K allows the user to watch upscaled anime in real time, as we believe in preserving original content and promoting freedom of choice for all anime fans. Re-encoding anime into 4K should be avoided as it is non-reversible, potentially damages original content by introducing artifacts, takes up to O(n2) more disk space and more importantly, does so without any meaningful decrease in entropy (lost information is lost).

Disclaimer: All art assets used are for demonstration and educational purposes. All rights are reserved to their original owners. If you (as a person or a company) own the art and do not wish it to be associated with this project, please contact us at anime4k.upscale@gmail.com and we will gladly take it down.

anime4K专为使用h.264, h.265或VC-1编码的原生1080p动画进行了优化，

使用方法

可以自行查看作者所写的英文指导https://github.com/bloc97/Anime4K/blob/master/md/GLSL_Instructions_Windows.md

以下均针对windows系统

下载anime4K

前往release界面下载最新文件并解压得到一堆GLSL文件

github一直出于被墙与能直连之间反复横跳的交叉状态，无法访问可使用魔法

下载并安装mpv播放器

作者原话:

If you wish to use another media player, look at their documentation on how to install GLSL shaders and modify the shader accordingly if needed.

大部分其他常用播放器（例如potplayer）并不支持GLSL的着色器

访问mpv官网

官网提供了各个系统的数种安装方式

（这里安利一下windows的版管理工具chocolatey,使用choco install mpv即可安装mpv）

添加着色器

将全部GLSL文件复制到%AppData%\mpv\shaders即可

编辑配置文件

在%AppData%\mpv下新建2个文件input.conf与mpv.conf（可选，根据这篇文章编写配置https://wiki.archlinux.org/title/Mpv#Configuration）

作者提供了针对不同显卡性能的3种预设方案

选择一种复制到input.conf即可

高端显卡

(例如 GTX 1080, RTX 2070, RTX 3060, RX 590, Vega 56, 5700XT, 6600XT)

如果输出分辨率不是4K，那么将占用更少GPU性能，也就是说，可以尝试更高级的配置方案

展开隐藏内容

入门显卡

(例如 GTX 980, GTX 1060, RX 570)

展开隐藏内容

使用方法

首先是mpv的使用方法

推荐使用命令行mpv 视频地址即可，支持url，但如果url中包含非英文字符，需要先手动对url进行解码

进入播放后，anime4K有3种模式,A,B,C.

分别对应快捷键ctrl+1/4,ctrl+2/5,ctrl+3/6.

1,2,3是一层，对性能要求更低

4,5,6是两层，对性能要求较高

使用ctrl+0关闭anime4K
